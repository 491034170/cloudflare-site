## 用途 / What it is

- 在线预览 / Live demo: <https://491034170.github.io/cloudflare-site/>

一个极简的 Cloudflare Pages 静态站点起步模板,用于快速部署:

- 🏠 个人主页 / 作品集
- 🎯 品牌落地页
- 🎉 临时活动页 / H5

A minimal starter for deploying static sites on Cloudflare Pages — perfect for personal pages, brand landings, or one-off campaigns.

## 快速部署 / Deploy

1. **Fork** 本仓库 → [Cloudflare Dashboard](https://dash.cloudflare.com) → **Pages** → 连接 Git 仓库
2. Build command 留空,Output directory 设为 `/`
3. 点击 **Deploy**,几秒钟后拿到 `*.pages.dev` 域名
4. (可选)绑定自定义域名,免费 SSL 自动签发

## 技术栈 / Stack

- 纯 HTML / CSS / JS,**零构建、零依赖**
- 部署于 **Cloudflare Pages**(免费额度足够个人/小团队使用)
- 支持自定义域名 + 全球 CDN
